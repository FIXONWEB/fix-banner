== Description ==

O plugin “fix-banner” é um pequeno trecho de códigos ou plugin do WordPress que nasceu da necessidade de poder postar uma imagem geralmente no formato de anuncio em vários locais do site  sem ter que recorrer a plugins pesados que disponibilizam um monte de funcionalidades mas que quase sempre não atende ao que queremos exatamente.

Trata-se de um tipo de postagem, portanto, um CPT do WordPress (ou Custom Post Type). Então, podemos definir que é uma forma de postar uma imagem e permitir que tal imagem possa ser exibida em outros locais do site. As imagens podem ser do tipo png, jpg ou até mesmo gif (animado ou não).

Para começar a funcionar, basta criar um post do tipo “banner“, definir uma imagem destacada, em seguida definir onde a imagem (do post) será exibida. isso é feito ao definir a categoria “Banner-Point”.

== Installation ==

1. Baixe o plugin do repositorio do Github
1. Instale o plugin indicando onde está o pacote zipado que acabou de baixar
1. Ative o plugin Fix-banner
1. Cadastre um ou mais banner, crie uma categoria
1. Na página ou no post, cole o shortcode [fix_banner set_local="__cateroria__"]

Substitua a palavra __cateroria__ pela categoria que vocẽ criou.
